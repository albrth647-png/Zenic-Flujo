🔗 Acceso:
  1. Obtén la URL del servicio:
     kubectl get ingress -n {{ .Release.Namespace }} {{ include "zenic-flujo.fullname" . }}

  2. O bien, forward del puerto local:
     kubectl port-forward -n {{ .Release.Namespace }} svc/{{ include "zenic-flujo.fullname" . }} 8080:8080

  3. Abre http://localhost:8080 en tu navegador

🔐 Configuración inicial requerida:
  1. Accede a la web UI y configura:
     - Licencia (Settings → License)
     - Usuario admin (por defecto: admin / admin)
     - Conectores en el Marketplace

📊 Monitoreo:
