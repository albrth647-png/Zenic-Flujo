# Frontend Utils

## Files: `frontend/src/lib/`, `frontend/src/utils/`, `frontend/src/locales/`
Frontend utility functions and i18n support.

### Utilities
- **lib/utils.ts** - shadcn/ui utility (cn() class merging)
- **utils/humanize.ts** - Human-readable formatting
- **locales/** - Translation files
- **i18n.ts** - i18n configuration

### Features
- Tailwind class merging (clsx + tailwind-merge)
- Date/currency formatting
- String humanization
- Multi-language support (ES, EN)

### Key Files
- `lib/utils.ts` - cn() utility
- `utils/humanize.ts` - Format helpers
- `i18n.ts` - i18n setup
