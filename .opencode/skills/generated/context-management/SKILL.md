# Context Management

## Session State
- Session data stored in `.swarm/` (auto-generated by runtime)
- Plans persisted in `.omo/plans/`
- Backups in `.omo/backups/`

## Backup Procedure
```bash
mkdir -p .omo/backups/$(date +%Y%m%d)
cp ~/.config/opencode/opencode.jsonc .omo/backups/$(date +%Y%m%d)/
```

## Restore
- Restore config: copy from `.omo/backups/<date>/` to `~/.config/opencode/`
- Always validate JSON after restore: `python3 -m json.tool <file>`

## Best Practices
- Backup before any config change
- Keep plans in `.omo/plans/` with `.md` extension
- Use checkpoints for multi-step work
